# effective-octo-plot

# Getting Started
First, install dependencies and activate venv in server.
```
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

For Python virtual environement. (localhost:5000)
```
python3 -m venv venv
source venv/bin/activate
```

Run the development server (localhost:5173).
```
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
